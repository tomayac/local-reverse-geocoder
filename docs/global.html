Type Definitions

callback(points, maxResults) → {Array.<object>}

Local reverse geocoder based on GeoNames data.
Parameters:
Name Type Description
points object | Array.<object> One single or an array of latitude/longitude pairs
maxResults integer The maximum number of results to return
Author:
  • Thomas Steiner (tomac@google.com)
License:
  • Apache 2.0
Source:
  • geocoder.js, line 1
Returns:
An array of GeoNames-based geocode results
Type
Array.<object>
Example
// With just one point
var point = {latitude: 42.083333, longitude: 3.1};
geocoder.lookUp(point, 1, function(err, res) {
  console.log(JSON.stringify(res, null, 2));
});

// In batch mode with many points
var points = [
  {latitude: 42.083333, longitude: 3.1},
  {latitude: 48.466667, longitude: 9.133333}
];
geocoder.lookUp(points, 1, function(err, res) {
  console.log(JSON.stringify(res, null, 2));
});
